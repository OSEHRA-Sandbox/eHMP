ehmp_synapse Cookbook
====================
This is a wrapper for the synapse cookbook.  It serves to keep project specific attributes and logic outside of the synapse cookbook.



Requirements
------------

#### packages
- `synapse` - ehmp_synapse needs the synapse cookbook.


Contributing
