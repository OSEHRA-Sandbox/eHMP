nerve_wrapper Cookbook
====================
This is a wrapper for the nerve cookbook.  It serves to keep project specific attributes and logic outside of the nerve cookbook.



Requirements
------------

#### packages
- `nerve` - nerve_wrapper needs the nerve cookbook.
